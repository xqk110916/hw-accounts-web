# 查看位置图接口文档

## 基本信息

| 项目 | 内容 |
|------|------|
| **接口路径** | `/api/busin/locationMap/positionMap` |
| **请求方法** | POST |
| **标签** | 位置地图管理 |
| **摘要** | 查看位置图 |
| **操作ID** | positionMapUsingPOST |

---

## 请求参数

**Content-Type**: `application/json`

| 字段 | 类型 | 必填 | 描述 |
|------|------|------|------|
| nodeId | integer | 否 | 节点ID |
| nodeType | string | 否 | 节点类型（1:平衡区 2:库房 3:排 4:行 5:位） |

### nodeType 枚举值

| 值 | 描述 |
|----|------|
| 1 | 平衡区 |
| 2 | 库房 |
| 3 | 排 |
| 4 | 行 |
| 5 | 位 |

---

## 响应数据

**Content-Type**: `application/json`

返回数组 `Array<LocationPositionVO>`

### LocationPositionVO

| 字段 | 类型 | 描述 |
|------|------|------|
| id | integer | 主键ID |
| warehouseId | integer | 库房ID |
| warehouseName | string | 库房名称 |
| balanceAreaId | integer | 平衡区ID |
| balanceAreaName | string | 平衡区名称 |
| shelfId | integer | 货架ID |
| shelfCode | string | 货架编号 |
| rowId | integer | 行ID |
| rowCode | string | 行编号 |
| columnId | integer | 列ID |
| columnCode | string | 列编号 |
| hierarchyId | integer | 关联层级表ID（列级节点ID） |
| status | integer | 状态（0:空闲 1:占用 2:锁定） |
| goodsId | integer | 货物ID |
| goodsCode | string | 货物编码 |
| goodsName | string | 货物名称 |
| goodsQuantity | number | 货物数量 |
| materialTypes | string | 材料类型（JSON数组格式） |
| grossWeight | string | 毛重 |
| netWeight | string | 净重 |
| tareWeight | string | 皮重 |
| weightUnit | string | 重量单位（kg、g等） |
| inboundOperationId | integer | 入库操作ID |
| initialId | integer | 初始操作ID |
| taskNum | string | 入库操作表的任务编号 |
| lastInboundTime | string | 最后入库时间 |
| lastOutboundTime | string | 最后出库时间 |
| createTime | string | 创建时间 |
| createUid | integer | 创建人ID |
| createUname | string | 创建人名称 |
| updateTime | string | 修改时间 |
| updateUid | integer | 修改人ID |
| updateUname | string | 修改人名称 |

---

## 响应状态码

| 状态码 | 描述 |
|--------|------|
| 200 | OK |
| 201 | Created |
| 401 | Unauthorized |
| 403 | Forbidden |
| 404 | Not Found |
