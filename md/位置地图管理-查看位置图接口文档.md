# 接口文档：查看位置图

## 基本信息

| 项目 | 内容 |
|------|------|
| **接口名称** | 查看位置图 |
| **接口路径** | `POST /locationMap/positionMap` |
| **Swagger路径** | `/位置地图管理/positionMapUsingPOST` |
| **Tag** | 位置地图管理 |
| **是否弃用** | 否 |

---

## 请求参数

### LocationStatisticDTO

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| `nodeId` | Long | 是 | 节点ID |
| `nodeType` | String | 是 | 节点类型（1:平衡区 2:库房 3:排 4:行 5:位） |

**请求示例：**
```json
POST http://10.10.41.179:40000/api/busin/locationMap/positionMap
Content-Type: application/json

{
  "nodeId": 1,
  "nodeType": "1"
}
```

---
