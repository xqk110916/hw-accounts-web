## 接口列表

### 1. 新增材料代码

- **接口地址**: `POST /api/busin/material-code/add`
- **接口描述**: 新增材料代码

**请求参数 (MaterialCodeDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| goodCode | string | 是 | 材料代码 |
| goodName | string | 是 | 材料名称 |
| commonName | string | 否 | 通用名称 |
| commonUnit | string | 否 | 常用单位 |
| elements | List | 否 | 元素及同位素含量列表 |

**elements 子字段 (ElementContentDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| element | string | 是 | 元素名称 |
| isotope | string | 否 | 同位素 |
| content | string | 否 | 含量 |

**请求示例**
```json
{
  "goodCode": "C001",
  "goodName": "铀235",
  "commonName": "浓缩铀",
  "commonUnit": "kg",
  "elements": [
    {
      "element": "U",
      "isotope": "235",
      "content": "3%"
    }
  ]
}
```

**响应示例**
```json
{
  "code": 1,
  "data": true,
  "msg": "操作成功",
  "success": true
}
```

### 4. 分页查询材料代码

- **接口地址**: `GET /api/busin/material-code/list`
- **接口描述**: 分页查询材料代码列表

**请求参数**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| currentPage | long | 是 | 当前页 |
| pageSize | long | 是 | 每页条数 |
| goodCode | string | 否 | 材料代码 |
| goodName | string | 否 | 材料名称 |
| commonName | string | 否 | 通用名称 |

**响应示例**
```json
{
  "code": 1,
  "data": {
    "list": [
      {
        "id": 1,
        "goodCode": "C001",
        "goodName": "铀235",
        "commonName": "浓缩铀",
        "commonUnit": "kg",

      "pageSize": 10,

    }
```

## 数据结构定义

### MaterialCodeDTO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | 主键ID（更新时使用） |
| goodCode | string | 材料代码 |
| goodName | string | 材料名称 |
| commonName | string | 通用名称 |
| commonUnit | string | 常用单位 |
| elements | List<ElementContentDTO> | 元素及同位素含量列表 |

### MaterialCodeVO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | 主键ID |
| goodCode | string | 材料代码 |
| goodName | string | 材料名称 |
| commonName | string | 通用名称 |
| commonUnit | string | 常用单位 |
| elements | List<ElementContentDTO> | 元素及同位素含量列表 |
| createTime | string | 创建时间 |
| createUid | long | 创建人ID |
| createUname | string | 创建人姓名 |
| updateTime | string | 更新时间 |
| updateUid | long | 更新人ID |
| updateUname | string | 更新人姓名 |

### ElementContentDTO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| element | string | 元素名称 |
| isotope | string | 同位素 |
| content | string | 含量 |
