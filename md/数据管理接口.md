## 一、材料代码管理

### 1. 新增材料代码

- **接口地址**: `POST /api/busin/material-code/add`

**请求参数 (MaterialCodeDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| goodCode | string | 是 | 材料代码 |
| goodName | string | 是 | 材料名称 |
| commonName | string | 否 | 通用名称 |
| commonUnit | string | 否 | 常用单位 |
| elements | List | 否 | 元素及同位素含量列表 |

**elements 子字段 (ElementContentDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| element | string | 是 | 元素名称 |
| isotope | string | 否 | 同位素 |
| content | string | 否 | 含量 |

**请求示例**
```json
{
  "goodCode": "C001",
  "goodName": "铀235",
  "commonName": "浓缩铀",
  "commonUnit": "kg",
  "elements": [
    {
      "element": "U",
      "isotope": "235",
      "content": "3%"
    }
  ]
}
```

**响应示例**
```json
{
  "code": 1,
  "data": true,
  "msg": "操作成功",
  "success": true
}
```

- **接口地址**: `POST /api/busin/label/data/add`

| rowSet | string | 否 | 单双排 |

{
  "templateId": 1,

  "remark": "测试标签",
  "dataJson": [
    {
      "fileName": "材料名称",
      "value": "铀235",
      "fontSize": "14",
