## 一、出库管理

### 1.1 提交出库申请

**接口地址**: `POST /outbound/submit`

**标签**: 出库管理

**请求参数 (OutboundSubmitDTO)**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| taskNum | string | 是 | 任务编号 |
| transferId | integer | 是 | 调拨依据ID |
| securityLevel | string | 是 | 密级 |
| outboundMan | string | 是 | 出库人 |
| receiveUnit | string | 否 | 收方单位 |
| warehouseName | string | 否 | 库房名称 |
| goodsList | array | 是 | 货物列表 |
| goodCodes | string | 否 | 材料编码，多个用逗号分隔 |
| goods | string | 否 | 材料，多个用逗号分隔 |
| submitType | integer | 是 | 暂存：4 ,提交：0 |
| remark | string | 否 | 备注 |

**货物列表项 (OutboundGoodsDTO)**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| containerCode | string | 是 | 容器号 |
| goodCode | string | 是 | 材料编码 |
| warehouseId | integer | 否 | 库房ID |
| warehouseName | string | 否 | 库房名称 |
| boxNum | string | 否 | 货箱号 |
| grossWeight | string | 否 | 毛重 |
| netWeight | string | 否 | 净重 |
| tareWeight | string | 否 | 皮重 |
| metalPercentage | string | 否 | 金属量% |
| productionUnit | string | 否 | 生产单位 |
| shelfCode | string | 否 | 排 |
| rowCode | string | 否 | 行 |
| columnCode | string | 否 | 列 |
| sealCode1 | string | 否 | 封记编码1 |
| sealCode2 | string | 否 | 封记编码2 |
| sealType1 | string | 否 | 封记类型1 |
| sealType2 | string | 否 | 封记类型2 |

**响应**: `boolean`

### 1.2 分页查询出库管理列表

**接口地址**: `POST /outbound/list`

**标签**: 出库管理

**请求参数 (OutboundManageQueryDTO)**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| currentPage | integer | 是 | 当前页 |
| pageSize | integer | 是 | 每页条数 |
| taskNum | string | 否 | 任务编号 |
| transferBasis | string | 否 | 调拨依据 |
| warehouseName | string | 否 | 库房 |
| receiveUnit | string | 否 | 收方单位 |
| containerCode | string | 否 | 容器号 |
| productionUnit | string | 否 | 生产单位 |
| startTime | string | 否 | 开始时间 |
| endTime | string | 否 | 结束时间 |
| statusList | array | 否 | 状态列表 |

**状态列表说明**:
- 0：待审核
- 1：审核通过
- 2：审核拒绝
- 7：变更审核中
- 8：变更审核通过
- 9：变更审核拒绝

**响应 (PageListVo«OutboundManageVO»)**:

| 参数 | 类型 | 说明 |
|------|------|------|
| list | array | 出库管理列表 |
| pagination | object | 分页信息 |

**OutboundManageVO**:

| 参数 | 类型 | 说明 |
|------|------|------|
| id | integer | ID |
| taskNum | string | 任务编号 |
| transferBasis | string | 调拨依据 |
| warehouseName | string | 库房 |
| receiveUnit | string | 收方单位 |
| auditStatus | integer | 审核状态 |
| dataStatus | integer | 操作记录状态 |
| statusDesc | string | 状态描述 |
| createTime | string | 出库时间 |

## 二、取货方案

### 2.1 自动配重

**接口地址**: `POST /pick/auto-weight`

**标签**: 取货方案

**说明**: 根据条件自动计算最优取货方案

**请求参数 (PickPlanRequestDTO)**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| materials | array | 是 | 物料列表（至少1个） |
| priorityWarehouse | boolean | 否 | 优先同一个库房 |
| priorityManufacturer | boolean | 否 | 优先同一个厂商 |
| priorityBatch | boolean | 否 | 优先同一个批次 |
| priorityFifo | boolean | 否 | 优先先进先出 |
| maxSolutions | integer | 否 | 返回最大方案数（默认3） |

**物料条件 (PickMaterialDTO)**:

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| goodCode | string | 否 | 材料编码 |
| materialName | string | 否 | 物料名称 |
| netWeight | number | 否 | 净重 |
| grossWeight | number | 否 | 毛重 |
| tareWeight | number | 否 | 皮重 |
| unit | string | 否 | 单位（如kg、g等） |
| suppliers | array | 否 | 出方单位列表（多选） |
| warehouses | array | 否 | 库房列表（多选） |
| batchNos | array | 否 | 任务批次列表（多选） |
| startStorageTime | string | 否 | 入库开始时间 |
| endStorageTime | string | 否 | 入库结束时间 |
| startDeliveryTime | string | 否 | 出库开始时间 |
| endDeliveryTime | string | 否 | 出库结束时间 |

**响应 (PickPlanVO)**:

| 参数 | 类型 | 说明 |
|------|------|------|
| solutionId | integer | 方案ID |
| priority | integer | 优先级排序 |
| containerCount | integer | 容器数量 |
| batchCount | integer | 涉及批次数量 |
| manufacturerCount | integer | 涉及厂商数量 |
| warehouseCount | integer | 涉及库房数量 |
| totalNetWeight | number | 总净重 |
| totalGrossWeight | number | 总毛重 |
| totalTareWeight | number | 总皮重 |
| oldestStorageTime | string | 最早入库时间 |
| containers | array | 容器列表 |

**容器信息 (ContainerInfoDTO)**:

| 参数 | 类型 | 说明 |
|------|------|------|
| containerCode | string | 容器号 |
| taskNum | string | 入库任务号 |
| batchNo | string | 批次号 |
| goodCode | string | 材料编码 |
| material | string | 物料名称 |
| warehouse | string | 库房 |
| shelfCode | string | 货架编号 |
| rowCode | string | 行编号 |
| columnCode | string | 列编号 |
| boxNo | string | 货箱号 |
| netWeight | number | 净重 |
| grossWeight | number | 毛重 |
| tareWeight | number | 皮重 |
| metalPercentage | string | 金属量% |
| manufacturer | string | 厂商 |
| productionUnit | string | 生产单位 |
| storageTime | string | 入库时间 |
| serialNumber | integer | 序号 |
| sealCode1 | string | 封记编码1 |
| sealCode2 | string | 封记编码2 |

### 2.2 手工选择

**接口地址**: `POST /pick/manual-pick`

**标签**: 取货方案

**说明**: 手动选择容器生成出库清单

**请求参数**:

```json
{
  "容器号列表": ["containerCode1", "containerCode2"]
}
```

**响应**: `object` - 返回出库清单结果
