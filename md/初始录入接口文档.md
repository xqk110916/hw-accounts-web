## 接口列表

### 1. 列表查询

- **接口地址**: `POST /api/busin/initial/list`
- **接口描述**: 列表查询初始录入记录

**请求参数 (InitialEntryQueryDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| currentPage | long | 是 | 当前页 |
| pageSize | long | 是 | 每页条数 |
| dataStatus | integer | 否 | 状态 |
| operationType | string | 否 | 操作类型 |
| startTime | string | 否 | 开始时间 |
| endTime | string | 否 | 结束时间 |

**请求示例**
```json
{
  "currentPage": 1,
  "pageSize": 10,
  "dataStatus": 0
}
```

**响应示例**
```json
{
  "code": 1,
  "data": {
    "records": [
      {
        "id": 1,
        "operationType": "INITIAL_ENTRY",
        "dataStatus": 0,
        "createTime": "2026-04-22 10:00:00",
        "createUid": 1,
        "createUname": "张三",
        "updateTime": "2026-04-22 10:00:00",
        "updateUid": 1,
        "updateUname": "张三"
      }
    ],
    "total": 100,
    "current": 1,
    "size": 10,
    "pages": 10
  },
  "msg": "操作成功",
  "success": true
}
```

### 2. 详情查询

- **接口地址**: `GET /api/busin/initial/detail`
- **接口描述**: 查询初始录入详情

**请求参数**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|

**响应示例**
```json
{
  "code": 1,
  "data": {
    "id": 1,
    "operationType": "INITIAL_ENTRY",
```

### 7. 导入

- **接口地址**: `POST /api/busin/initial/import`
- **接口描述**: 导入初始录入数据（Excel）

**请求参数**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| file | file | 否 | 导入文件 |
| dataId | long | 否 | 导入数据id（重新导入时此id必填） |

**响应示例**
```json
{
  "code": 1,
  "data": {
    "successCount": 10,
    "failCount": 0,
    "totalCount": 10,
    "dataId": "123456789",
    "failRows": [],
    "taskNum": "CSR20260422001"
  },
  "msg": "操作成功",
  "success": true
}
```

**返回字段说明**

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| successCount | integer | 成功条数 |
| failCount | integer | 失败条数 |
| totalCount | integer | 总条数 |
| dataId | string | 导入数据ID |
| failRows | List | 失败行信息 |
| taskNum | string | 任务编号 |

## 枚举值参考

### 数据状态 (dataStatus / status)
| 值 | 含义 |
|----|------|
| 0 | 待确认 |
| 1 | 已确认 |
| 2 | 待提交 |
| 3 | 审核中 |
| 4 | 审核通过 |
| 5 | 审核驳回 |
