## 接口列表

### 1. 新增文号

- **接口地址**: `POST /api/busin/doc-number/add`
- **接口描述**: 新增文号

**请求参数 (DocNumberDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| docCode | string | 是 | 文号编码 |
| docName | string | 是 | 文号名称 |

**请求示例**
```json
{
  "docCode": "WH-2026-001",
  "docName": "测试文号"
}
```

**响应示例**
```json
{
  "code": 1,
  "data": true,
  "msg": "操作成功",
  "success": true
}
```

### 5. 分页查询文号列表

- **接口地址**: `GET /api/busin/doc-number/list`
- **接口描述**: 分页查询文号列表

**请求参数**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| pageNum | integer | 否 | 页码（默认1） |
| pageSize | integer | 否 | 每页条数（默认10） |
| docCode | string | 否 | 文号编码 |
| docName | string | 否 | 文号名称 |

**响应示例**
```json
{
  "code": 1,
  "data": {
    "records": [
      {
        "id": 1,
        "docCode": "WH-2026-001",
        "docName": "测试文号1",
        "createTime": "2026-04-22 10:00:00",
        "createUname": "张三"
      },
      {
        "id": 2,
        "docCode": "WH-2026-002",
        "docName": "测试文号2",
        "createTime": "2026-04-21 10:00:00",
        "createUname": "李四"
      }
    ],
    "total": 100,
    "current": 1,
    "size": 10,
    "pages": 10
  },
  "msg": "操作成功",
  "success": true
}
```

## 数据结构定义

### DocNumberDTO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | ID（更新时使用） |
| docCode | string | 文号编码 |
| docName | string | 文号名称 |

### DocNumberVO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | ID |
| docCode | string | 文号编码 |
| docName | string | 文号名称 |
| createTime | string | 创建时间 |
| createUname | string | 创建人 |
| updateTime | string | 更新时间 |
| updateUname | string | 更新人 |
