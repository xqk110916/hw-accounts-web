## 接口列表

### 1. 新增调拨依据

- **接口地址**: `POST /api/busin/transfer-basis/add`
- **接口描述**: 新增调拨依据

**请求参数 (TransferBasisDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| documentNo | string | 是 | 文号 |
| type | string | 是 | 类型（0：调入、1：调出） |
| goodsList | List | 否 | 调拨货物列表 |
| status | integer | 否 | 状态（0：未完成 1：已完成） |

**goodsList 子字段 (TransferGoods)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| goodCode | string | 是 | 货物编码 |
| goodNum | string | 否 | 批准件数 |
| goodWeight | string | 否 | 批准量 |

**请求示例**
```json
{
  "documentNo": "DB-2026-001",
  "type": "0",
  "goodsList": [
    {
      "goodCode": "C001",
      "goodNum": "10",
      "goodWeight": "100kg"
    }
  ]
}
```

**响应示例**
```json
{
  "code": 1,
  "data": true,
  "msg": "操作成功",
  "success": true
}
```

## 数据结构定义

### TransferBasisDTO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | 主键id，修改时必填 |
| documentNo | string | 文号 |
| type | string | 类型（0：调入、1：调出） |
| status | integer | 状态（0：未完成、1：已完成） |
| goodsList | List<TransferGoods> | 调拨货物列表 |

### TransferBasisVO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | 主键ID |
| documentNo | string | 文号 |
| type | string | 类型（0：调入、1：调出） |
| status | integer | 状态（0：未完成、1：已完成） |
| goodsList | List<TransferGoodsVO> | 调拨货物列表 |
| createTime | string | 创建时间 |
| createUid | string | 创建人ID |
| createUname | string | 创建人姓名 |
| updateTime | string | 更新时间 |
| updateUid | string | 更新人ID |
| updateUname | string | 更新人姓名 |

### TransferBasisQueryDTO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| pageNum | integer | 页码 |
| pageSize | integer | 每页大小 |
| documentNo | string | 文号 |
| type | string | 类型（0：调入、1：调出） |
| status | integer | 状态（0：未完成、1：已完成） |

### TransferGoods

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| goodCode | string | 货物编码 |
| goodNum | string | 批准件数 |
| goodWeight | string | 批准量 |

### TransferGoodsVO

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | 主键ID |
| transferBasisId | long | 调拨依据ID |
| goodCode | string | 货物编码 |
| goodNum | string | 批准件数 |
| goodWeight | string | 批准量 |
