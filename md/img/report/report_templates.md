# 报表图片字段与布局样式描述文档

本文档对 `md/img/report/` 目录下的 7 张报表图片（R01、R03、R04、R05、R06、R08、R09）进行了详细的字段提取和布局样式描述。

---

## 统一布局特点 (Common Layout Features)
除 R01 之外，R03, R04, R05, R06, R08, R09 均遵循高度一致的**两栏式网格对称布局**，具有以下共同点：
- **页眉区域**：
  - 左上角第一行：密级选择（下拉框，默认值为“内部”）。
  - 左上角第二行：表号（例如：表号：X材料R03表）。
  - 中间居中：大标题（一等字号，粗体，例如：X材料库存变化统计报表）。
  - 右上角第二行：格式（例如：格式：X管办94R03号）。
- **主体表格（左右两栏对称）**：
  - **左半栏**：由多行输入项组成，每行由一个固定标签（Label）和一个或多个输入框（Input）/日期选择框组成，带有完整边框。
  - **右半栏**：
    - 第一行：“负责人（签字）” 标签 + 空白签名区。
    - 第二行：“制表人（签字）” 标签 + 空白签名区。
    - 剩余行合并大单元格：“单位领导签名和单位盖章” 标签 + 空白盖章区（右下角标有“年月日”）。
- **页脚区域（表格外底部）**：
  - 采用水平一横行排版，包含三个固定文案：
    - 左侧：批准机关（国家统计局）
    - 中间：批准文号（XXX字 (xxxx) xxx号）
    - 右侧：制表机关（X材料XX办公室）

表格采用左右对称的两大板块，分别代表“发方”和“收方”，中间以粗线/双线分隔：

##### 【左半板块 —— 发方】
- **发方主体信息（第1、2行组合）**：
  - 左侧包含一个垂直跨行单元格，文本为“发方”。
  - 右侧分为上下两行：
    - 上行：标签为“单位名称” + 输入框（占位符：“系统预设可修改”）。

- **发方流程与签名区（第3至6行组合）**：

    - 第一行：标签“发货日期” + 日期选择器（占位符：“日期选择”）。
    - 第二行：标签“发货地点” + 输入框（占位符：“请输入”）。
    - 第三行：标签“发货人（签名）” + 签名空白区。
    - 第四行：标签“核查人（签名）” + 签名空白区（右下角带“年月日”）。
  - 右侧为一个合并的大单元格：
    - 标签为“发方单位领导签名和单位盖章”，下方为大片空白区，右下角带“年月日”。

##### 【右半板块 —— 收方】
- **收方主体信息（第1、2行组合）**：
  - 左侧包含一个垂直跨行单元格，文本为“收方”。
  - 右侧分为上下两行：
    - 上行：标签为“单位名称” + 下拉选择框（占位符：“选择，后返现单位代号，许可证号”）。
    - 下行：横向均分两半，左边是“单位代号”标签 + 输入框（占位符：“请输入”）；右边是“许可证号”标签 + 输入框（占位符：“请输入”）。
- **收方流程与签名区（第3至6行组合）**：

    - 第一行：标签“收货日期” + 日期选择器（占位符：“日期选择”）。
    - 第二行：标签“收货地点” + 输入框（占位符：“请输入”）。
    - 第三行：标签“收货人（签名）” + 签名空白区。
    - 第四行：标签“核收入（签名）” + 签名空白区（右下角带“年月日”）。
  - 右侧为一个合并的大单元格：

#### 1.3 底部附加信息
- **报表编号行**：
