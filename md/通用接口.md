## 一、批次号生成

### 1. 生成批次号

- **接口地址**: `GET /api/busin/batchNo/generate`
- **接口描述**: 生成指定类型的批次号，格式为年+3位序列号

**请求参数**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| batchType | string | 是 | 批次类型（入库/出库: 1; 位置移动: 2; 实物盘存: 4; 其他累加） |
| prefix | string | 否 | 前缀（目前不传） |

**响应示例**
```json
{
  "code": 1,
  "data": "2026001",
  "msg": "操作成功",
  "success": true
}
```

**返回说明**
- 格式：`年份 + 3位序列号`（如：2026001）
- 如果传入了 `prefix`，格式为：`前缀 + 年份 + 3位序列号`（如：RK2026001）

**使用示例**
```
GET /api/busin/batchNo/generate?batchType=0
Response: "2026001"

GET /api/busin/batchNo/generate?batchType=inbound&prefix=RK
Response: "RK2026001"
```

---

### 2. 重置序列号

- **接口地址**: `POST /api/busin/batchNo/reset`
- **接口描述**: 重置指定类型和年份的序列号

**请求参数**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| batchType | string | 是 | 批次类型 |
| year | integer | 否 | 年份（可选，不传则重置当前年份） |

**响应示例**
```json
{
  "code": 1,
  "data": {},
  "msg": "操作成功",
  "success": true
}
```

**使用示例**
```
POST /api/busin/batchNo/reset?batchType=inbound&year=2026
```

## 二、Excel导入

### 1. 导入Excel文件

- **接口路径**: `POST /api/busin/excelImport/import`
- **接口说明**: 支持xls和xlsx两种格式

**请求参数**

| 参数名 | 说明 | 类型 | 必填 |
|--------|------|------|------|
| file | 文件 | file | 是 |
| importType | 导入类型：inbound（入库）、outbound（出库） | string | 是 |
| goodCodes | 材料编码，多个用`,`分隔 | string | 否 |
| inboundImportType | 入库导入类型（1-2），只针对入库操作：1-信息导入1、2-信息导入2 | integer | 否 |
| sheetIndex | sheet索引，从0开始 | integer | 否 |
| sheetName | sheet名称 | string | 否 |
| taskNum | 任务编号 | string | 否 |

**响应结果**

返回 `ImportResultVO` 对象
