## 2. 新增字典项

- 请求方式: `POST`
- 请求地址: `/base/dictionary`
- Content-Type: `application/json`

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| parentId | String | 是 | 父级ID，顶层传 "0"，子级传对应父级ID |
| fullName | String | 是 | 字典名称 |
| dictValue | String | 是 | 字典值 |
| bizCode | String | 是 | 业务编码 |
| sortNum | Number | 否 | 排序号，默认 99 |
| enableFlag | Number | 否 | 启用状态：1-启用 |
| description | String | 否 | 描述 |
| dictType | String | 是 | 字典类型，固定值 "category" |

### 请求示例

```json
{
  "parentId": "0",
  "fullName": "字典名称",
  "dictValue": "zdmc",
  "bizCode": "zdmc",
  "sortNum": 99,
  "enableFlag": 1,
  "description": "",
  "dictType": "category"
}
```

### 响应示例

```json
{
  "code": 1,
  "data": null,
  "msg": "操作成功",
  "success": true
}
```

## 4. 修改字典项

- 请求方式: `PUT`
- 请求地址: `/base/dictionary/{id}`
- Content-Type: `application/json`

### 请求参数

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | String | 是 | 字典项ID |
| parentId | String | 是 | 父级ID |
| parentName | String | 否 | 父级名称 |
| fullName | String | 是 | 字典名称 |
| dictValue | String | 是 | 字典值 |
| bizCode | String | 是 | 业务编码 |
| sortNum | Number | 否 | 排序号 |
| enableFlag | Number | 否 | 启用状态：1-启用 |
| description | String | 否 | 描述 |
| dictType | String | 是 | 字典类型，固定值 "category" |
| color | String | 否 | 颜色 |
| defaultFlag | Number | 否 | 是否默认 |

### 请求示例

```json
{
  "id": "2046423912768405505",
  "parentId": "0",
  "parentName": "",
  "fullName": "测试",
  "dictValue": "ceshi",
  "bizCode": "ceshi",
  "sortNum": 99,
  "enableFlag": 1,
  "description": "",
  "dictType": "category",
  "color": null,
  "defaultFlag": null
}
```

### 响应示例

```json
{
  "code": 1,
  "data": null,
  "msg": "操作成功",
  "success": true
}
```
