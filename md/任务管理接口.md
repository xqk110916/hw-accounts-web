# 任务管理接口文档

## 基础信息

- **Base URL**: `http://10.10.41.179:40000/api/busin`
- **Swagger 地址**: `http://10.10.41.179:40000/api/busin/doc.html`
- **API 分组**: `1.X`

---

## 实物盘存

### 1. 分页查询实物盘存列表

**接口地址**: `POST /inventory/page/list`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| currentPage | integer (int64) | ✅ | 当前页 |
| pageSize | integer (int64) | ✅ | 每页条数 |
| inventoryTime | string (date-time) | ❌ | 盘存时间 |
| taskNum | string | ❌ | 任务编号（模糊查询） |
| warehouseName | string | ❌ | 库房名称（模糊查询） |

**响应**: `PageListVo«PhysicalInventoryManageVO»`

**PhysicalInventoryManageVO**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | integer (int64) | 任务ID |
| taskNum | string | 任务编号 |
| inventoryTime | string (date-time) | 盘存时间 |
| status | integer (int32) | 状态码 |
| statusDesc | string | 状态描述 |
| warehouseCount | integer (int32) | 库房数量 |
| warehouseNames | string | 库房名称（逗号分隔） |
| createTime | string (date-time) | 创建时间 |

---

### 2. 根据选择的仓库获取货物清单（生成任务）

**接口地址**: `POST /inventory/goods/list`

**请求参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| type | string | ✅ | 类型（all:全部仓库，selected:选择仓库） |
| warehouseId | array[string] | ❌ | 选择模式下的仓库列表 |

**响应**: `array[PhysicalInventoryGoodsVO]`

**PhysicalInventoryGoodsVO**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | integer (int64) | 主键id |
| warehouseId | integer (int64) | 库房Id |
| warehouseName | string | 库房名称 |
| containerCode | string | 容器号 |
| goodCode | string | 材料代码 |
| goodName | string | 材料名称 |
| productionUnit | string | 生产单位 |
| location | string | 位置 |
| sealCode1 | string | 封记编号1 |
| sealCode2 | string | 封记编号2 |
| storageTime | string (date-time) | 入库时间 |
| result | string | 结果 0:正常 1:盘亏 2:盘盈 |
| remark | string | 备注 |

---

### 3. 提交暂存任务

**接口地址**: `POST /inventory/submit`

**请求参数** (`PhysicalInventorySubmitDTO`):

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| taskNum | string | ✅ | 任务编号 |
| selectType | string | ✅ | 类型（1:全部仓库，2:选择仓库） |
| submitType | integer (int32) | ✅ | 提交类型：0-提交（待审核），4-暂存（待提交） |
| warehouseIds | string | ❌ | 库房id列表（多个用逗号分隔） |
| warehouseNames | string | ❌ | 库房名称列表（多个用逗号分隔） |
| remark | string | ❌ | 备注 |
| goodsList | array[PhysicalInventoryGoodsDTO] | ❌ | 货物清单 |

**PhysicalInventoryGoodsDTO**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| warehouseId | integer (int64) | 库房id |
| warehouseName | string | 库房名称 |
| containerCode | string | 容器号 |
| goodCode | string | 材料代码 |
| goodName | string | 材料名称 |
| productionUnit | string | 生产单位 |
| location | string | 位置（货架-行-列） |
| sealCode1 | string | 封记编号1 |
| sealCode2 | string | 封记编号2 |
| storageTime | string (date-time) | 入库时间 |
| result | string | 结果（0:正常 1:盘亏 2:盘盈） |
| resultRemark | string | 结果备注 |

**响应**: `boolean`

---

### 4. 录入盘存结果

**接口地址**: `POST /inventory/result/submit`

**请求参数** (`PhysicalInventoryDTO`):

| 参数名 | 类型 | 必填 | 说明 |

| id | integer (int64) | ✅ | 主键 |
| taskNum | string | ✅ | 任务编号 |
| wareList | array[PhysicalInventoryWareHoseDTO] | ❌ | 库房请单 |

**PhysicalInventoryWareHoseDTO**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| warehouseId | integer (int64) | 库房id |
| warehouseName | string | 库房名称 |
| inventoryMan | string | 盘存人 |
| inventoryTime | string (date-time) | 盘存时间 |
| superviseMan | string | 监盘人 |
| responsiblePerson | string | 负责人 |
| statusInspector | string | 封记检查人 |
| resultStatus | string | 盘存结果 1：全部正常 2：部分异常 |
| normalCount | integer (int32) | 正常数 |
| deficitCount | integer (int32) | 盘亏数 |
| deficitRemark | string | 盘亏备注 |
| excessCount | integer (int32) | 盘盈数 |
| excessRemark | string | 盘盈备注 |
| goodsList | array[PhysicalInventoryGoodsDTO] | 货物清单 |

**响应**: `boolean`

### 5. 查询盘存任务详情

**接口地址**: `GET /inventory/detail/{id}`

**路径参数**:

| 参数名 | 类型 | 必填 | 说明 |
|--------|------|------|------|
| id | integer (int64) | ✅ | 任务ID |

**响应**: `PhysicalInventoryDetailVO`

**PhysicalInventoryDetailVO**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| inventory | PhysicalInventoryEntity | 盘存任务信息 |
| warehouseList | array[PhysicalInventoryWarehouseVO] | 库房列表（含货物清单） |
| totalNormalCount | integer (int32) | 总正常数 |
| totalDeficitCount | integer (int32) | 总盘亏数 |
| totalExcessCount | integer (int32) | 总盘盈数 |

**PhysicalInventoryEntity**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | integer (int64) | ID |
| taskNum | string | 任务编号 |
| selectType | string | 类型 |
| warehouseIds | string | 库房id（多个用逗号分隔） |
| warehouseNames | string | 库房名称（多个用逗号分隔） |
| remark | string | 备注 |
| dataStatus | integer (int32) | 数据状态（0:待审核 1:审核通过 2:审核拒绝 4:待提交） |
| auditRemark | string | 审核备注 |
| auditTime | string (date-time) | 审核时间 |
| auditUserId | integer (int64) | 审核人ID |
| auditUserName | string | 审核人名称 |
| createTime | string (date-time) | 创建时间 |
| createUid | integer (int64) | 创建人ID |
| createUname | string | 创建人名称 |
| updateTime | string (date-time) | 更新时间 |
| updateUid | integer (int64) | 更新人ID |
| updateUname | string | 更新人名称 |

**PhysicalInventoryWarehouseVO**:

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | integer (int64) | id |
| warehouseId | integer (int64) | 库房ID |
| warehouseName | string | 库房名称 |
| inventoryMan | string | 盘存人 |
| inventoryTime | string (date-time) | 盘存时间 |
| superviseMan | string | 监盘人 |
| responsiblePerson | string | 负责人 |
| statusInspector | string | 封记检查人 |
| resultStatus | string | 盘存结果（1:全部正常 2:部分异常） |
| normalCount | integer (int32) | 正常数 |
| deficitCount | integer (int32) | 盘亏数 |
| deficitRemark | string | 盘亏备注 |
| excessCount | integer (int32) | 盘盈数 |
| excessRemark | string | 盘盈备注 |
| goodsList | array[PhysicalInventoryGoodsVO] | 该库房货物清单 |
