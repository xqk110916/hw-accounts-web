## 一、入库管理

### 1.1 分页查询入库管理列表

- **接口地址**: `POST /api/busin/inbound/manage/pageList`

**请求参数 (InboundManageQueryDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| currentPage | long | 是 | 当前页 |
| pageSize | long | 是 | 每页条数 |
| taskNum | string | 否 | 任务编号 |
| warehouseName | string | 否 | 库房 |
| outUnit | string | 否 | 出方单位 |
| productionUnit | string | 否 | 生产单位 |
| transferBasis | string | 否 | 调拨依据 |
| containerCode | string | 否 | 容器号 |
| startTime | string | 否 | 开始时间 |
| endTime | string | 否 | 结束时间 |
| statusList | List\<Integer\> | 否 | 状态列表 |

**响应示例**
```json
{
  "code": 1,
  "data": {
    "list": [
      {
        "id": 1,
        "taskNum": "RK20260422001",
        "warehouseName": "xxx库房",
        "productionUnit": "xxx单位",
        "transferBasis": "xxx调拨依据",
        "auditStatus": 0,
        "dataStatus": 0,
        "createTime": "2026-04-22 10:00:00"
      }
    ],
    "pagination": {
      "currentPage": 1,
      "pageSize": 10,
      "total": 100
    }
  },
  "msg": "操作成功",
  "success": true
}
```

**返回字段说明 (InboundManageVO)**

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | ID |
| taskNum | string | 任务编号 |
| warehouseName | string | 库房名称 |
| outUnit | string | 出方单位 |
| transferBasis | string | 调拨依据 |
| auditStatus | integer | 审核状态 |

| createTime | string | 创建时间 |

## 三、取货方案

### 3.1 自动配重

- **接口地址**: `POST /pick/auto-weight`
- **说明**: 根据条件自动计算最优取货方案

**请求参数 (PickPlanRequestDTO)**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| materials | array | 是 | 物料列表（至少1个） |
| priorityWarehouse | boolean | 否 | 优先同一个库房 |
| priorityManufacturer | boolean | 否 | 优先同一个厂商 |
| priorityBatch | boolean | 否 | 优先同一个批次 |
| priorityFifo | boolean | 否 | 优先先进先出 |
| maxSolutions | integer | 否 | 返回最大方案数（默认3） |

**物料条件 (PickMaterialDTO)**

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| goodCode | string | 否 | 材料编码 |
| materialName | string | 否 | 物料名称 |
| netWeight | number | 否 | 净重 |
| grossWeight | number | 否 | 毛重 |
| tareWeight | number | 否 | 皮重 |
| unit | string | 否 | 单位（如kg、g等） |
| suppliers | array | 否 | 出方单位列表（多选） |
| warehouses | array | 否 | 库房列表（多选） |
