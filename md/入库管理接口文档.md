## 入库管理

### 1. 分页查询入库管理列表

- **接口地址**: `POST /api/busin/inbound/manage/pageList`
- **接口描述**: 分页查询入库管理列表

**请求参数 (InboundManageQueryDTO)**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|
| currentPage | long | 是 | 当前页 |
| pageSize | long | 是 | 每页条数 |
| taskNum | string | 否 | 任务编号 |
| warehouseName | string | 否 | 库房 |
| outUnit | string | 否 | 出方单位 |
| productionUnit | string | 否 | 生产单位 |
| transferBasis | string | 否 | 调拨依据 |
| containerCode | string | 否 | 容器号 |
| startTime | string | 否 | 开始时间 (格式: date-time) |
| endTime | string | 否 | 结束时间 (格式: date-time) |
| statusList | List<Integer> | 否 | 状态列表 |

**响应示例**
```json
{
  "code": 1,
  "data": {
    "list": [
      {
        "id": 1,
        "taskNum": "RK20260422001",
        "warehouseName": "xxx库房",
        "productionUnit": "xxx单位",
        "transferBasis": "xxx调拨依据",
        "auditStatus": 0,
        "dataStatus": 0,
        "createTime": "2026-04-22 10:00:00"
      }
    ],
    "pagination": {
      "currentPage": 1,
      "pageSize": 10,
      "total": 100
    }
  },
  "msg": "操作成功",
  "success": true
}
```

**返回字段说明 (InboundManageVO)**

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | ID |
| taskNum | string | 任务编号 |
| warehouseName | string | 库房 |
| productionUnit | string | 单位 |
| transferBasis | string | 调拨依据 |
| auditStatus | integer | 审核状态 |
| dataStatus | integer | 操作记录状态 |
| createTime | string | 入库时间 |

### 8. 根据节点类型查询节点列表（查询库房）

**路径参数**

| 参数名称 | 类型 | 必填 | 描述 |
|---------|------|------|------|

**响应示例**

**返回字段说明 (LocationHierarchyVO)**

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | 主键ID |
| nodeName | string | 节点名称 |
| nodeCode | string | 节点编号 |
| nodeType | integer | 节点类型（1:平衡区 2:库房 3:货架 4:行 5:列） |
| balanceAreaId | long | 平衡区ID |

| warehouseId | long | 库房ID |

### 10.根据任务号和容器号查询入库货物

## 基本信息

| 项目 | 内容 |
|------|------|
| **接口名称** | 根据任务号和容器号查询入库货物 |
| **接口路径** | `GET /inbound/getByTaskNumAndContainerCode` |
| **Swagger路径** | `/入出库管理/getByTaskNumAndContainerCodeUsingGET` |
| **Tag** | 入库管理 |
| **是否弃用** | 否 |

---

## 请求参数

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| `taskNum` | String | 是 | 任务号 |

**请求示例：**
```
GET http://10.10.41.179:40000/api/busin/inbound/getByTaskNumAndContainerCode?taskNum=XXX&containerCode=XXX
```
