# 节点详情页说明文档

## 一、页面概述
- 展示单个节点的详细信息及库存使用情况
- 分为「节点基本信息」和「库存使用量」两大模块

---

## 二、节点基本信息

### 功能说明
- 展示节点的基础属性信息
- 支持点击「编辑」按钮进入编辑模式修改信息

### 字段说明

| 字段名 | 说明 | 是否必填 |
|--------|------|----------|
| 节点 | 节点唯一标识编码 | 是 |
| 名称 | 节点的业务名称 | 是 |
| 调入许可证 | 物资调入该节点所需的许可证编号 | 是 |
| 调出许可证 | 物资调出该节点所需的许可证编号 | 是 |
| 备注 | 节点的补充说明信息 | 否 |
| 类型 | 节点存储类型，支持「本地」与「代存」两种模式切换 | 是 |

---

## 三、库存使用量

### 功能说明
- 以环形图可视化展示库存占用比例
- 右侧列表展示各材料的具体使用量
- 支持点击材料项进行选中高亮，图表与列表联动

### 统计指标

| 指标 | 说明 |
|------|------|
| 总库位 | 该节点的总库存容量 |
| 已使用 | 当前已占用的库位数量 |
| 使用率 | 已使用库位占总库位的百分比（环形图中心展示） |

### 材料库存明细

| 字段 | 说明 |
|------|------|
| 材料名称 | 各材料的名称标识 |
| 使用量 | 该材料当前占用的库位数量 |
| 选中状态 | 当前选中的材料项高亮显示，图表同步联动 |

---

## 四、交互说明

| 交互 | 说明 |
|------|------|
| 类型切换 | 开关组件可在「本地」与「代存」之间切换节点类型 |
| 编辑功能 | 点击右上角「编辑」按钮，进入节点信息编辑状态 |
| 材料选中 | 点击右侧材料列表项，该项高亮，图表同步联动反馈 |
