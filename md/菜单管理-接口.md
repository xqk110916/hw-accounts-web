# 菜单管理接口文档

> [!NOTE]
> 在新增业务界面时，若涉及新目录或新功能页，需同步调用相关接口完成菜单注册，以确保导航系统正常显示。
> 示例流程：若新增模块 `task`，需先调用新增目录接口（类型 M），获取 ID 后，再在其下调用新增菜单接口（类型 C）。

---

## 1. 查询菜单树
获取当前系统中已注册的菜单结构。

- **请求方式**: `GET`
- **请求地址**: `/base/menu/treeListByClientType`

### 请求参数
| 参数名 | 类型 | 必选 | 描述 | 示例值 |
| :--- | :--- | :--- | :--- | :--- |
| `clientType` | String | 是 | 客户端类型 | `0` |

### 返回示例
```json
{
  "code": 1,
  "data": [
    {
      "id": "1782971139194662913",
      "pid": "0",
      "sortNum": 1,
      "label": "应用管理",
      "fullName": "应用管理",
      "parentId": "0",
      "enCode": "application",
      "path": "application",
      "frameFlag": null,
      "menuType": "M",
      "visibleFlag": "1",
      "perms": "",
      "icon": "component",
      "clientType": "0",
      "remark": null,
      "sysFlag": 1,
      "linkType": 0,
      "linkUrl": null,
      "children": [ {}, {}, {} ]
    }
  ]
}
```

## 2. 新增菜单 (目录/界面)
向系统中添加新的菜单项。

- **请求方式**: `POST`
- **请求地址**: `/base/menu`

### 请求体参数 (JSON)
| 字段名 | 类型 | 描述 | 注意事项 |
| :--- | :--- | :--- | :--- |
| `parentId` | String | 父级菜单 ID | 顶级菜单请填 `0` |
| `menuType` | String | 菜单类型 | `M`: 目录, `C`: 菜单 |
| `fullName` | String | 菜单显示文本 | - |
| `linkType` | Number | 链接类型 | `0`: 内部页面, `1`: 外部页面 (默认 0) |
| `path` | String | 界面对应目录 | 对应代码中的文件夹名称 |
| `linkUrl` | String | 外部链接 URL | 仅在 `linkType` 为 1 时有效 |
| `perms` | String | 权限唯一标识 | - |
| `sortNum` | Number | 排序权重 | 数值越小越靠前 |
| `visibleFlag` | String | 显示状态 | `0`: 隐藏, `1`: 显示 |
| `clientType` | String | 客户端类型 | 固定传 `0` |
| `icon` | String | 图标名称 | - |
| `enCode` | String | 英文编码 | - |

### 请求体示例
```json
{
  "parentId": "2042412186163056641",
  "menuType": "C",
  "fullName": "测试界面",
  "linkType": 0,
  "path": "test",
  "linkUrl": "",
  "perms": "test",
  "sortNum": 99,
  "visibleFlag": "1",
  "clientType": "0",
  "icon": "",
  "enCode": "csjm"
}
```

## 3. 修改菜单
更新现有菜单的信息。

- **请求方式**: `PUT`
- **请求地址**: `/base/menu/${id}`

### 请求体示例
```json
{
  "id": "2044665620060798978",
  "parentId": "2042412186163056641",
  "menuType": "C",
  "fullName": "测试界面1",
  "path": "test",
  "perms": "test",
  "enCode": "csjm1",
  "sortNum": 99,
  "visibleFlag": "1",
  "clientType": "0",
  "icon": "",
  "linkType": 0,
  "linkUrl": "",
  "sysFlag": 0,
  "children": null,
  "remark": null,
  "frameFlag": null
}
```

## 4. 查询菜单详情
获取指定菜单的详细配置。

- **请求方式**: `GET`
- **请求地址**: `/base/menu/${id}`

### 请求参数
| 参数名 | 类型 | 必选 | 描述 |
| :--- | :--- | :--- | :--- |
| `id` | String | 是 | 菜单 ID |

### 返回示例
```json
{
  "code": 1,
  "data": {
    "id": "2044665620060798978",
    "parentId": "2042412186163056641",
    "menuType": "C",
    "fullName": "测试界面1",
    "path": "test",
    "perms": "test",
    "enCode": "csjm1",
    "sortNum": 99,
    "visibleFlag": "1",
    "clientType": "0",
    "icon": "",
    "linkType": 0,
    "linkUrl": "",
    "sysFlag": 0,
    "children": null,
    "remark": null,
    "frameFlag": null
  },
  "msg": "操作成功",
  "success": true
}
```

## 5. 删除菜单
从系统中移除指定的菜单项。

- **请求方式**: `DELETE`
- **请求地址**: `/base/menu/${id}`

### 请求参数
| 参数名 | 类型 | 必选 | 描述 |
| :--- | :--- | :--- | :--- |
| `id` | String | 是 | 菜单 ID |

### 返回体示例
```json
{
  "code": 1,
  "data": null,
  "msg": "操作成功",
  "success": true
}
```
