# 库房管理接口文档

**Base URL:** `http://10.10.41.179:40000/api/busin`

---

## 一、位置地图管理

### 1.1 层级节点管理

#### 1.1.1 查询所有节点（树形结构）

- **接口地址**: `GET /locationMap/hierarchy/tree`

---

#### 1.1.2 根据平衡区ID查询树形结构

- **接口地址**: `GET /locationMap/hierarchy/treeByBalanceArea/{balanceAreaId}`

**路径参数**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| balanceAreaId | Long | 是 | 平衡区ID |

---

#### 1.1.3 分页查询节点列表

- **接口地址**: `POST /locationMap/hierarchy/pageList`

**请求参数**

```json
{
  "currentPage": 1,
  "pageSize": 10
}
```

---

#### 1.1.4 根据父节点ID查询子节点列表

- **接口地址**: `GET /locationMap/hierarchy/listByParentId/{parentId}`

**路径参数**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| parentId | Long | 是 | 父节点ID |

---

#### 1.1.5 根据节点类型查询节点列表

- **接口地址**: `GET /locationMap/hierarchy/listByNodeType/{nodeType}`

**路径参数**

| 参数名 | 类型 | 必填 | 描述 |
|--------|------|------|------|
| nodeType | Integer | 是 | 节点类型（1:平衡区 2:库房 3:货架 4:行 5:列） |

**响应示例**
```json
{
  "code": 1,
  "data": [
    {
      "id": 1,
      "nodeName": "库房1",
      "nodeCode": "KF001",
      "nodeType": 2,
      "balanceAreaId": 1,
      "balanceAreaName": "平衡区1",
      "warehouseType": "0",
      "warehouseCode": "KF001",
      "warehouseName": "库房1",
      "remark": "备注",
      "createTime": "2026-04-22 10:00:00"
    }
  ],
  "msg": "操作成功",
  "success": true
}
```

**返回字段说明 (LocationHierarchyVO)**

| 参数名称 | 类型 | 描述 |
|---------|------|------|
| id | long | 主键ID |
| nodeName | string | 节点名称 |
| nodeCode | string | 节点编号 |
| nodeType | integer | 节点类型（1:平衡区 2:库房 3:货架 4:行 5:列） |
| balanceAreaId | long | 平衡区ID |
| balanceAreaName | string | 平衡区名称 |
| warehouseId | long | 库房ID |
| warehouseName | string | 库房名称 |
| warehouseCode | string | 库房编号 |
| warehouseType | string | 库房类型 (0:新库, 1:老库) |
| remark | string | 备注 |

---

#### 1.1.6 查询节点详情

**路径参数**

## 附录：枚举值参考

### 节点类型 (nodeType)
| 值 | 说明 |
|:---:|------|
| 1 | 平衡区 |
| 2 | 库房 |
| 3 | 货架 |
| 4 | 行 |
| 5 | 列 |

### 位置状态 (status)
| 值 | 说明 |
|:---:|------|
| 0 | 空闲 |
| 1 | 占用 |
| 2 | 锁定 |

### 库房类型 (warehouseType)
| 值 | 说明 |
|:---:|------|
| 0 | 新库 |
| 1 | 老库 |

### 平衡区类型 (type)
| 值 | 说明 |
|:---:|------|
| 本地 | 本地 |
| 代存 | 代存 |

### 数据状态 (dataStatus / status)
| 值 | 含义 |
|----|------|
| 0 | 待确认 |
| 1 | 已确认 |
| 2 | 待提交 |
| 3 | 审核中 |
| 4 | 审核通过 |
| 5 | 审核驳回 |
