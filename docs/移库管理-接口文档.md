# 移库管理 - 接口文档

> **Base URL**: `http://10.10.41.179:40000/api/busin`

---

## 接口列表

| 序号 | 方法 | 路径 | 说明 |
|------|------|------|------|
| 1 | POST | `/move/list` | 分页查询移库管理列表 |
| 2 | GET | `/move/detail/{id}` | 查询移库详情 |
| 3 | POST | `/move/submit` | 提交移库申请 |
| 4 | POST | `/move/update` | 编辑移库任务 |
| 5 | POST | `/move/confirm` | 审核移库申请 |
| 6 | POST | `/move/executeAudited` | 执行审核后的更新 |
| 7 | GET | `/move/cancelApply` | 取消修改申请 |

## 1. 分页查询移库管理列表

**接口地址**: `POST /move/list`

**说明**: 分页查询移库管理列表，支持按任务编号、容器号、位置、状态、时间等条件筛选。

**请求参数** (Body - JSON):

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| currentPage | integer (int64) | ✅ | 当前页 |
| pageSize | integer (int64) | ✅ | 每页条数 |
| taskNum | string | ❌ | 任务编号 |
| containerCode | string | ❌ | 容器号 |
| position | string | ❌ | 位置（原位置或目标位置模糊查询） |
| statusList | integer[] | ❌ | 状态列表（0-待审核，1-已审核，2-审核驳回，4-暂存） |
| startTime | string (date-time) | ❌ | 开始时间 |
| endTime | string (date-time) | ❌ | 结束时间 |

**请求示例**:
```json
{
  "currentPage": 1,
  "pageSize": 10,
  "taskNum": "TASK001",
  "statusList": [0, 1]
}
```

**响应参数** (`PageListVo<MoveManageVO>`):

| 参数 | 类型 | 说明 |
|------|------|------|
| list | array | 数据列表 |
| list[].id | integer (int64) | ID |
| list[].moveTaskNum | string | 任务编号 |
| list[].auditStatus | integer (int32) | 审核状态 |
| list[].dataStatus | integer (int32) | 操作记录状态 |
| list[].containerCount | integer (int32) | 容器数量 |
| list[].createTime | string (date-time) | 创建时间 |
| list[].moveTime | string (date-time) | 移动时间 |
| list[].statusDesc | string | 状态描述 |
| pagination | object | 分页信息 |
| pagination.currentPage | integer (int32) | 当前页 |
| pagination.pageSize | integer (int32) | 每页条数 |
| pagination.total | integer (int32) | 总条数 |

## 3. 提交移库申请

| goodsList[].containerCode | string | ❌ | 容器号 |
| goodsList[].goodCode | string | ❌ | 材料编码 |
| goodsList[].productionUnit | string | ❌ | 生产单位 |
| goodsList[].goodsWeight | string | ❌ | 物料重量 |
| goodsList[].grossWeight | string | ❌ | 毛重 |
| goodsList[].netWeight | string | ❌ | 净重 |
| goodsList[].tareWeight | string | ❌ | 皮重 |
| goodsList[].sealCode1 | string | ❌ | 封记编码1 |

| goodsList[].sealType1 | string | ❌ | 封记类型1 |

| goodsList[].sourceWarehouse | string | ❌ | 原库房 |
| goodsList[].sourceShelf | string | ❌ | 原排 |
| goodsList[].sourceRow | string | ❌ | 原行 |

## 5. 审核移库申请

**接口地址**: `POST /move/confirm`

**说明**: 对移库申请进行审核，支持通过或驳回。

**请求参数** (Body - JSON, `MoveConfirmDTO`):

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| taskNum | string | ✅ | 任务编号 |
| approved | boolean | ✅ | 是否通过 |
| remark | string | ❌ | 审核备注 |

**请求示例**:
```json
{
  "taskNum": "TASK001",
  "approved": true,
  "remark": "审核通过"
}
```

**响应**: `boolean` — true 表示成功，false 表示失败。

## 6. 执行审核后的更新

**接口地址**: `POST /move/executeAudited`

**说明**: 根据审核结果执行实际的移库数据更新。审核通过后调用此接口生效。

**请求参数** (Query String):

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| approved | boolean | ✅ | 是否通过 |
| modifyRecordId | integer (int64) | ✅ | 修改记录ID |
| remark | string | ❌ | 备注 |

**请求示例**:
```
POST /move/executeAudited?approved=true&modifyRecordId=100&remark=执行移库
```

**响应**: `boolean` — true 表示成功，false 表示失败。

---

## 7. 取消修改申请

**接口地址**: `GET /move/cancelApply`

**说明**: 取消正在审核中的修改申请。

**请求参数** (Query String):

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| modifyId | integer (int64) | ❌ | 修改记录ID |

**请求示例**:
```
GET /move/cancelApply?modifyId=100
```

**响应**: 成功返回空响应。
